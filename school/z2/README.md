# Magic Sum Calculator

A C++ program that generates a random matrix of integers and calculates the "Magic Sum" - the sum of all odd numbers containing the digit 3.

The program prompts the user to enter the desired matrix size, generates random numbers between 1 and 100, displays the matrix, and then computes the magic sum by finding all odd numbers that contain the digit 3.

# Калькулятор Магической Суммы

Программа на C++, которая генерирует случайную матрицу целых чисел и вычисляет "Магическую сумму" - сумму всех нечетных чисел, содержащих цифру 3.

Программа запрашивает у пользователя желаемый размер матрицы, генерирует случайные числа от 1 до 100, отображает матрицу, а затем вычисляет магическую сумму, находя все нечетные числа, содержащие цифру 3.
